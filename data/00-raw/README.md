This is the code we used to get our data. There are instructions on how to use it in the readme in the github.
https://github.com/Lucien2714/leetify-data-crawler
We used two different rating systems while finding data on players. Using CS2 Rating while finding data on average players is perfect because
there are rarely cheaters that play around this rating. However, when filtering for high rated players, there are rarely regular players that
play and most of the games are just filled with cheaters. Instead, we use Faceit ranking which is what every good player uses and there are
no cheaters.
